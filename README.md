Простой бот для тг на гемини 2.5 flash, отвечает если в сообщении есть имя: Скаперс; Скаперсон; Cкаперсик
Есть модули на погоду (погода [название города]), голосовые сообщения из текста (/voice [текст], реакции (ставит сам, рандомно, шанс можно поменять в файле reactions.js), бан людей чтобы бот им не отвечал (/ban [id], /unban [id]), память (без понятия как хорошо она работает сейчас, требует доработки!!!) а также консоль которая работает у тех, чей айди указан в ADMINS= (вызывается просто написав консоль)
Чтобы запустить, должен быть node js, для начала добавьте в .env токен телеграмм, гемини апи ключ и т.д, промпт уже есть но если хотите - меняйте как душе угодно (желательно поменять то на какие имена он отзывается в bot.js!!!)
После перейдите в папку бота в терминале (cd путь_к_папке), пропишите npm install, после завершения также в папке бота node bot.js, если все указано верно, вас оповестят о том что бот запущен (BOT STARTED [id])
